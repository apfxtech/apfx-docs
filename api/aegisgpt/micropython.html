AegisGPT v2.1 - Что это?

AegisGPT - это мощный и гибкий API-интерфейс для генерации текста на основе искусственного интеллекта. AegisGPT использует передовые технологии глубокого обучения, такие как GPT-3, 4, 5 и Transformer-XL, чтобы создавать высококачественный, креативный и уникальный контент для различных целей и задач. интерфейс использует разные библиотеки. Например: сервер AegisGPT, OpenAI, G4F и другие. Проект активно развиваеться, но некоторые функции недоступны. Оффициальным ботом можно воспользоваться в Telegram.

Полный пример:

Этот код является обработчиком использования API AegisGPT от ApertureFox, который позволяет генерировать текст на основе запросов пользователя. Код делает следующее:

  • В бесконечном цикле запрашивает у пользователя ввод текста, который сохраняет в переменной data["request"].
  • Отправляет этот текст в формате JSON на адрес https://aperturefox.ru/api/gpt, используя библиотеку requests.
  • Если сервер успешно обработал запрос и вернул код 200, то код получает ответ в формате JSON и разбирает его.
  • Выводит на экран сгенерированный текст, разбивая его по строкам, а также имя сервера, который обработал запрос, и время генерации текста.
  • Если сервер не смог обработать запрос и вернул другой код, то код выводит на экран ошибку с этим кодом.
MicroPython - пример использования AegisGPT API
import urequests as requests
# Импортируем модуль network для работы с беспроводными сетями
import network

# Создаем объект sta_if для подключения к точке доступа (station interface)
sta_if = network.WLAN(network.STA_IF)
# Активируем sta_if
sta_if.active(True) # Запускаем wifi
# Сканируем доступные сети и возвращаем список кортежей с информацией о каждой сети
sta_if.scan() # Сканируем сети
# Получаем ввод от пользователя для имени сети
lan = input("Имя сети: ")
# Получаем ввод от пользователя для пароля сети
pswd = input("Пароль сети: ")
# Подключаемся к сети с указанными именем и паролем
sta_if.connect(lan, pswd) # Подключение к сети
# Проверяем, подключены ли мы к сети, и возвращаем True или False
sta_if.isconnected() # Проверка на подключение

request_array = []
data = {
    "device": "MicroPythonTest", # Название устройства
    "server": "You", # Название сервера
    "version": "ChatGPT-AUTO", # Версия модели
    "request": [], # Список запросов
    "proxy": "None" # Прокси-сервер, если "None", то используется встроенный, но очень медленный сервер.
}

while(True):
    print("\n")
    request = {"role":"user", "content":input("Ваш запрос: ")} # создаем запрос в формате JSON
    request_array.append(request) # добавляем запрос в массив с контекстом
    data["request"] = request_array # сохраняем массив с контекстом в data
    print(request_array) # выводим массив для понимания из чего состоит запрос?
    print("\n")
    response = requests.post("https://aperturefox.ru/api/gpt", json=data) # Отправляем запрос
    if response.status_code == 200: # Если сервер ответил
        result = response.json() # парсим
        for i in result["response"].split("\n"): # Для удобного вывода в терминал, выводим каждую строку последовательно
            print(i) # печатаем
        print("\nПровайдер: " + str(result["server"])) # Выводим имя провайдера
        print("Время генерации: " + str(result["wrtime"])) # Выводим время генерации

        request = {"role":"assistant", "content": result["response"].replace("\n", " ")} # запоминаем ответ бота
        request_array.append(request) # добавляем ответ в массив с контекстом
        data["request"] = request_array # сохраняем массив с контекстом 

    else: # Если сервер не ответил
        print(f"Ошибка: {response.status_code}, {response.json()}") # Выводим ошибку

Внимание!

AegisGPT API поддерживает только POST запросы, если вы получаете статус сервера "500", проверьте отправку данных. Ошибка "500" - означает ошибку сервера, которая чаще всего происходит по вине ползователя.
